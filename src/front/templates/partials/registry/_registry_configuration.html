Unity Catalog schema — triples views & registry tables
This UC schema hosts the Delta Lake views that expose Knowledge Graph triples to SQL queries (e.g. triples, entities, relationships) and the Lakebase-backed registry tables that store domains, versions, permissions, and global configuration. The app service principal needs USE CATALOG, USE SCHEMA, and SELECT (plus CREATE TABLE / MODIFY for the registry).
—
Unity Catalog Volume — binary domain document uploads
This Unity Catalog managed Volume stores the binary artefacts uploaded through the ontology designer (PDF, Word, images in each domain's documents/ folder). It is not used for registry rows (those live in Lakebase). The app service principal needs READ VOLUME and WRITE VOLUME privileges to upload and serve document files.
